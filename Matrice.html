Matricea este un tablou bidimensional( adica este un vector pe un vector).
Ea are n linii si m coloane. De obicei indicile pentru linii este i si pentru coloane este j.














La fel ca si la vector, daca avem o matrice cu 100 de linii si 100 de coaloane, initializam matricea cu 1 in plus adica int a[101][101];
Daca numerele din matrice nu sunt intregi, vom initializa matricea float a[101][101]; (pentru numere reale).

Citirea si afisarea sunt esentiale pentru a putea lucra cu matrici. 

          int main()
          {
            int a[101][101],i,j,n,m;
            //Citirea
            cin>>n>>m;
            for(i=1; i<=n; i++)
              for(j=1; j<=m; j++)
                cin>>a[i][j];

            //Prelucram matricea

            //Afisarea

            for(i=1; i<=n; i++)
            {
                for(j=1; j<=m; j++)
                  cout<< a[i][j]<<" ";
                cout<< endl;
            }
          }

Pentru a determina suma de pe o anumita linie, luam toata linia cu indice (i=2 spre exemplu) si consideram linia ca si un vector. 

          int main()
          {
            int a[101][101], n, m, i, j, si=0, sc=0;
            cin>>n>>m;
            for(i=1; i<=n; i++)
              for(j=1; j<=m; j++)
                cin>>a[i][j];
            for(j=1; j<=m; j++)
              si=si+a[2][j];
            for(i=1; i<=m; i++)
              sc=sc+a[i][2];
            cout<<"Suma liniei 2 este "<< si;
            cout<<"Suma coloanei 2 este "<< sc;

            return 0;
          }

Într-o matrice pătratică numarul de linii= numarul de coloane (n=m).
Într-o matrice pătratică avem:

  • Diagonala principala elementele a[i][i], cu i=1,n sau a[i][i], cu i=0,n-1
  • Diagonala secundara elementele a[i][n-i+1], i=1,n sau a[i][n-i-1], i=0,n-1

Zonele determinate de diagonale:

I.
Pe diagonala principală i = j
Sub diagonala principala: i > j
Deasupra diagonalei principale: i < j
II.
Pe diagonala secundară: j + i = n + 1
Sub diagonala secundara: j + i > n + 1
Deasupra diagonalei secundare:j + i < n + 1

Se da o matrice patratica cu n linii si n coloane. Sa se umple matricea in felul urmator:
-In N sa se puna 0.
-In S sa se puna 2.
-In E sa se puna 3.
-In V sa se puna 4.
-Pe diagonala principala sa se puna 1, iar pe cea secundara sa se complteze cu -1. 

            int main()
            {
              int a[101][101], i, j, n;
              cin>>n;

              for(i=1; i<=n; i++)
                for(j=1; j<=n; j++)
                  {
                    if(i == j)
                        a[i][j] = 1; //Diagonala principala
                      else
                    if(i + j == n + 1)
                        a[i][j] = -1; //Diagonala secundara
                      else
                    if(i < j && j + i < n + 1)
                        a[i][j] = 0; //In N matricei
                      else
                    if(i > j &&  j + i > n + 1)
                        a[i][j] = 2; //In S matricei
                      else
                    if(i < j &&  j + i > n + 1)
                        a[i][j] = 3; //In E matricei
                      else
                    if(i > j &&  j + i < n + 1)
                        a[i][j] = 4; //In V matricei
                  }

                    for(i=1; i<=n; i++)
                      { for(j=1; j<=n; j++)
                        cout<< a[i][j]<<" ";
                      cout<< endl;
                    }
              return 0;
            }

Se da un numar natural n, sa se afiseze matricea in forma de spirala. Exemplu: date.in: n=4
date.out:
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7 

          
int main()
{
  int a[10][10],n,i,j,k,numar,x,y,v;
  cin>>n;
  k=1;
  numar=1;
  x=1;
  y=1;
  v=0;
  while (k <= n)
  {
    for (j=k; j<=n-k+1; j++)
    {
      a[x][j] = numar;
      ++numar;
    }
    y=--j;
    for (i=k+1; i<=n-k+1; i++)
    {
      a[i][y]=numar;
      ++numar;
    }
    x=--i;
    for(j=n-k; j>=k; j--)
    {
      a[x][j]=numar;
      ++numar;
    }
    y=++j;
    for (i=n-k; i>=k+1; i--)
    {
      a[i][y]=numar;
      ++numar;
    }
    x=++i;
    k++;
  }
  for(i=1; i<=n; i++)
  {
    for (j=1; j<=n; j++)
      cout<< a[i][j]<<"  ";
    cout<< endl;
  }
  return 0;
}
