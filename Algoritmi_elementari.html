 Operatorii aritmetici sunt : "+", "/", "-", "*", "%" .
           "A + B" reprezinta suma dintre A si B.
           "A / B" reprezinta catul impartirii lui A la B.
           "A - B" reprezinta diferenta dintre A si B.
           "A * B" reprezinta produsul dintre A si B.
           "A % B" reprezinta restul impartirii lui A la B.
                     ex: A%10 = ultima cifra a lui A.
                         A%2 = 0(A este numar par), 1(este numar impar)
       
 Operatorii relationali sunt: "<", ">", "!=", "<=", "=>", "==", "===" .
            "A < B" reprezinta ca numarul B e mai mare decat numarul A.
            "A > B" reprezinta ca numarul A e mai mare decat numarul B.
            "A != B" reprezinta ca numerele A si B sunt diferite.
            "A <= B" reprezinta ca numarul B este mai mare sau egal cu numarul A.
            "A => B" reprezinta ca numarul B este mai mic sau egal cu numarul A.
            "A == B" reprezinta ca numerele A si B sunt egale(le verifica valoarea numerica).
            "A === B" reprezinta ca numerele A si B sunt egale(numeric) si tipul lor(int, long long, char, etc ).
        
Operatorii logici sunt: "&&", "||", "!".
        
        Toti operatorii logici returneaza valori de tipul bool, true sau false
        
  • "A &&B" reprezinta doua conditii.
             ex: if(x < 10 &&x != 0){
                  // x respecta ambele conditiile 
                }
                else{
                  // x nu respecta ambele conditii
                 }
           
  •  "A || B" reprezinta ca una dintre cele doua conditii este adevarata.
             ex: if(A || B){
                   // este respectata una dintre conditii
                  }
                 else{
                   // nu este respectata nicio conditie
                 }
                 
           
  •  "!A" reprezinta negarea lui A.
             ex:if(x!=0)
                  // x este diferit de 0

Se citeste un numar n, sa se aifiseze suma cifrelor numarului n.
Exemplu: n=1045
s=10 


        int main()
        {
         int n,s=0; //declaram variabile pe care le vom folosi

         while(n!=0){
         s=s+n%10; //adunam la suma ultima cifra a numarului

         }

       }

Exemplu: n=8
6 12 9 1 2 3 6 19

         int main()
         {
          int n,mini,x,i;
          cin>>n;
          cin>>x;
          mini=x; //Minimul se initializeaza cu primul element sau cu cea mai mare valoare posibila
          for(i=1;i< n;i++)
            {
              cin>>x;
              if(x< mini)
                mini=x;
            }
          cout<<"Minimul este "<< mini;
         }
